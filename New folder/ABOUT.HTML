About the System

Guimaras Insight Engine is a web and mobile application designed to evaluate and showcase the products and service providers in the Province of Guimaras using AI-powered analysis. The system aims to support consumers, tourists, and stakeholders by providing data-driven insights on quality, authenticity, and overall performance.

System Objectives

  • Categorize services and product providers by location and industry for easy browsing and review.
  • Allow tourists and customers to make informed decisions based on real-time reviews and ratings..
  • Develop an AI-based analysis tool that summarizes customer sentiments and rates performance.
  • Generate and visualize analytical reports to improve the products and services.

Key Features

  • Category-based listing of products and service providers
  • AI-generated ratings and evaluation summaries
  • Interactive dashboard with evaluation statistics
  • User-friendly and responsive web interface
  • Offline-capable demonstration using client-side data storage

AI Evaluation Concept

The AI component of the Guimaras Insight Engine simulates intelligent decision-making by analyzing predefined quality indicators, ratings, and evaluation criteria. Although implemented without a live backend, the system demonstrates how AI-generated insights can be applied to real-world product and service assessment scenarios.

Academic Purpose

This system is developed as a Bachelor of Science in Information Systems Capstone Project. It demonstrates system analysis, design, implementation, and user experience principles, integrating modern web technologies and simulated AI functionality.
